# CV - Miguel López Pérez | Panadero

Curriculum interactivo y responsivo con funcionalidad de descarga a PDF.

## 🌟 Características

- ✨ **Diseño moderno y elegante** con paleta de colores cálida
- 📱 **Completamente responsivo** - Se adapta a cualquier dispositivo
- 📥 **Descarga a PDF** con un solo clic
- 🖨️ **Fallback a impresión nativa** si hay problemas de CORS
- 🎨 **Interfaz limpia** construida con Bootstrap 5
- ⚡ **Rápido y ligero** - Sin dependencias pesadas

## 🛠️ Tecnologías utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos personalizados y responsive
- **JavaScript Vanilla** - Funcionalidad de descarga
- **Bootstrap 5.3.3** - Framework CSS
- **Google Fonts** - Tipografía elegante
- **Font Awesome 6.4.0** - Iconografía
- **html2pdf.js** - Generación de PDF
- **WebP** - Imágenes optimizadas

## 📋 Contenidos

- **Perfil Profesional** - Descripción general
- **Áreas de Desempeño** - 4 especialidades principales:
  - Masa Dulce
  - Masas Madre
  - Hojaldres
  - Repostería Básica y Postres
- **Información de Contacto** - Teléfono, email, ubicación
- **Experiencia Laboral** - Timeline de empleos anteriores

## 🚀 Cómo usar

### Opción 1: Localmente en el navegador
1. Descarga o clona este repositorio
2. Abre `index.html` en tu navegador web
3. Haz clic en "Descargar CV" para exportar a PDF

### Opción 2: En un servidor
```bash
cd "tu-directorio"
python -m http.server 8000
```
Luego abre `http://localhost:8000` en tu navegador

## 📥 Descargar CV

El botón "Descargar CV" genera automáticamente un archivo PDF con el curriculum completo, optimizado para impresión.

**Nota:** Si ves advertencias en la consola sobre CORS, es normal. El programa cuenta con un fallback automático que abre la ventana de impresión nativa como alternativa.
